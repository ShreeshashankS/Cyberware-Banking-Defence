# **App Name**: BankSafe Simulator

## Core Features:

- Banking System Simulation: Simulate a simplified banking IT infrastructure, focusing on critical assets and interdependencies.
- Attack Vector Simulation: Simulate common cyber attacks, including ransomware, DDoS, data breaches, and supply chain attacks, each with customizable parameters.
- Defensive Measure Implementation: Enable players to implement various defensive measures (e.g. firewall rules, patching, employee training) and observe their impact on the bank's security posture.
- AI-driven Risk Recommendations: Suggest security control improvements based on analysis of threats, vulnerabilities, and the current infrastructure configuration; uses a tool to reason over available options to ensure relevance.
- Incident Response Simulation: Simulate real-time incident response scenarios, requiring players to make quick decisions under pressure to minimize damage.
- Performance Score Calculation: Display scores for different roles, enabling a competitive view of minimizing attacks, response time and financial impact.

## Style Guidelines:

- Primary color: Deep navy blue (#2E3148) to evoke trust and security.
- Background color: Light gray (#F0F2F5) for a clean, professional look.
- Accent color: Muted gold (#A3916A) for highlights and important actions.
- Body and headline font: 'Inter', a neutral, modern sans-serif, suitable for both headlines and body text.
- Use a set of consistent, minimalist icons for different types of assets, threats, and controls.
- Design a clean and organized dashboard layout that facilitates efficient monitoring and control.
- Subtle animations to highlight changes in the system, such as successful attacks or implemented defenses.
